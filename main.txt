# GenericDAO - Data Access Object Générique

## Description du projet
GenericDAO est une implémentation générique du pattern DAO (Data Access Object) en Java. Ce projet fournit une couche d'abstraction pour les opérations de base de données, permettant une gestion simplifiée et réutilisable des entités persistantes.

## Objectifs
- Fournir une interface générique pour les opérations CRUD
- Réduire la duplication de code dans les couches d'accès aux données
- Faciliter la maintenance et l'évolution des accès aux données
- Standardiser les opérations de persistance dans l'application

## Fonctionnalités principales
- Opérations CRUD génériques (Create, Read, Update, Delete)
- Gestion des transactions
- Support des requêtes personnalisées
- Manipulation d'entités via réflexion Java
- Gestion des erreurs et exceptions

## Technologies utilisées
- Java
- JPA (Java Persistence API)
- Hibernate
- Maven pour la gestion des dépendances
- JUnit pour les tests unitaires

## Structure du projet
- src/main/java : Code source principal
  - dao/ : Implémentation du DAO générique
  - entities/ : Classes d'entités
  - utils/ : Classes utilitaires
- src/test/java : Tests unitaires
