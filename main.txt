# Projet GenericDAO - Résumé

## Description du projet
GenericDAO est une implémentation générique du pattern DAO (Data Access Object) en Java. Ce projet fournit une couche d'abstraction pour les opérations de base de données, permettant une gestion simplifiée et standardisée des entités persistantes.

## Objectifs principaux
- Fournir une interface générique pour les opérations CRUD
- Réduire la duplication de code dans les couches d'accès aux données
- Faciliter la maintenance et l'évolution des applications
- Standardiser les opérations de persistance

## Structure du projet
- src/
  - dao/
    - GenericDAO.java (Interface générique)
    - GenericDAOImpl.java (Implémentation de base)
  - utils/
    - HibernateUtil.java (Utilitaire de configuration Hibernate)
  - entities/
    - Entités métier

## Technologies utilisées
- Java
- Hibernate (ORM)
- JPA (Java Persistence API)
- Maven (Gestion des dépendances)
- MySQL (Base de données)

## Instructions d'utilisation
1. Configuration de la base de données :
   - Créer une base de données MySQL
   - Configurer les paramètres de connexion dans hibernate.cfg.xml

2. Intégration dans un projet :
   - Étendre GenericDAOImpl pour vos entités spécifiques
   - Injecter les dépendances nécessaires
   - Utiliser les méthodes CRUD génériques
