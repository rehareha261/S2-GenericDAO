# GenericDAO - Data Access Object Générique

## Description du Projet
GenericDAO est une implémentation générique du pattern DAO (Data Access Object) en Java, conçue pour simplifier l'accès aux données dans les applications Java. Ce projet fournit une couche d'abstraction pour les opérations de base de données, rendant le code plus maintenable et réutilisable.

## Objectifs Principaux
- Fournir une interface générique pour les opérations CRUD
- Réduire la duplication de code dans les couches d'accès aux données
- Faciliter la maintenance et l'évolution des applications
- Standardiser l'accès aux données à travers l'application

## Structure du Projet
Le projet est organisé selon une architecture standard:
- src/
  - main/
    - java/
      - dao/         : Implémentations DAO
      - model/       : Classes modèles
      - util/        : Classes utilitaires
  - test/           : Tests unitaires

## Technologies Utilisées
- Java
- JPA (Java Persistence API)
- Hibernate
- MySQL/PostgreSQL (bases de données supportées)
- Maven (gestion des dépendances)
