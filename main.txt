# Résumé du Projet

Ce projet est une application de démonstration mettant en œuvre une interaction de base avec une base de données via un DAO générique. 
Il inclut une classe `Main` qui initialise une connexion à une base de données imaginaire et la constitue avec un exemple d'entité `Money`. 
Le projet comprend également des tests unitaires pour valider les fonctionnalités implémentées.

## Caractéristiques clés

- **Connexion à la base de données**: Utilisation d'une classe fictive `DBConnection` pour illustrer la connexion à une base de données.
- **DAO Générique**: La classe `GenericDAO` permet des opérations de base sur des entités génériques.
- **Entité `Money`**: Un exemple d'entité utilisée pour montrer l'interaction avec la base de données.
- **Tests Unitaires**: Implémentation de tests avec JUnit et Mockito pour assurer la validité du code.

## Objectifs du Projet

- Illustrer l'utilisation des DAO pour la gestion des données.
- Démontrer la manière d'implémenter des tests unitaires robustes.
- Fournir une base de code réutilisable pour des projets similaires.
